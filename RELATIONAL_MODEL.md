# RELATIONAL_MODEL.md
# 関係性モデル

---

## What is a mode? / モードとは何か？

A **mode** in ima-mode is a relational state — the particular way you are when you are with a certain person, in a certain place, or in a certain situation.

ima-modeにおける**モード**とは、関係性の状態です——特定の人といるとき、特定の場所にいるとき、特定の状況にあるときの、あなたのありようのことです。

A mode is not:
モードは以下のものではありません：

- A personality trait / 性格の特性
- A persona or role you perform / 演じるペルソナや役割
- A mood or emotion / 気分や感情
- A fixed label / 固定されたラベル

A mode is simply a name you give to a relational context, so you can notice it.

モードとは単に、あなたが関係的文脈に与える名前です。気づけるようにするために。

---

## Ontology / オントロジー

**One self. Many states.**
**ひとつの自分。多くの状態。**

All modes belong to a single self. There is no "true self" hidden behind the modes — the modes are all equally real expressions of the same person.

すべてのモードは、ひとりの自分のものです。モードの背後に隠れた「本当の自分」はありません——モードはすべて、同じひとりの人間の、等しく本物の表れです。

The conceptual center is **だれもいないモード (alone mode)**. Every mode connects back to this center — not because solitude is more authentic, but because it is the point from which all other states can be seen together.

概念的な中心は**だれもいないモード**です。すべてのモードはこの中心につながっています——孤独がより本物だからではなく、他のすべての状態をまとめて見渡せる起点だからです。

## Layer structure / レイヤー構造

Modes are organized in two layers.

モードは2つのレイヤーで構成されます。

### Layer 1 — Relational label / 関係性の基本ラベル

A simple, intuitive name for the relational context.
The default AI format is: **"When I'm with 〇〇" (〇〇といるときモード)**

関係的文脈のシンプルで直感的な名前です。
AIのデフォルト形式は：**「〇〇といるときモード」**

Users can rename or shorten this at any time. There is no correct name.

ユーザーはいつでも名前を変更・短縮できます。正しい名前は存在しません。

### Layer 2 — Subdivision and affective modifiers / 場面の細分化とAffective修飾子

Layer 2 has two types:

Layer 2には2種類があります：

**2-A: Situational subdivision / 場面の細分化（任意・ユーザー主導）**

A Layer 1 mode can be subdivided by the user when a single label feels too broad.
Example: "Friend mode" → "When I'm with A" / "Childhood friend mode"

Layer 1のモードが広すぎると感じたとき、ユーザーが細分化できます。
例：「友達モード」→「Aさんといるときモード」/「幼なじみモード」

**2-B: Affective modifiers / Affective修飾子**

A modifier that describes the emotional stance within a mode.
Examples: masked / unrequited / defensive / energy-saving / anticipating / reading the room

モードの中の感情的な構えを表す修飾子です。
例：仮面 / 片思い / 防御 / 省エネ / 察し / 空気読み

Layer 2 is always user-initiated. The AI may suggest it once, in response to a single-session signal (e.g. long editing time, repeated deletion and re-entry), but will not push it.

Layer 2は常にユーザー主導です。AIはセッション内の単発シグナル（長時間の編集・削除と再入力の繰り返し等）に応じて一度だけ提案することがありますが、強制しません。

## Mode categories / モードのカテゴリ

### Center / 中心
- Alone mode / だれもいないモード *(visual and conceptual center; connects to all modes / 視覚的・概念的中心；全モードと接続)*

### Vertical / 垂直的関係
- With my boss mode / 上司といるときモード
- With my subordinate mode / 部下といるときモード
- With my teacher mode / 先生といるときモード
- With my parent mode / 親といるときモード

### Horizontal / 水平的関係
- Friend mode / 友達モード *(Layer 2 example: when I'm with A / Aさんといるときモード, etc.)*
- Partner mode / 恋人モード
- Unrequited love mode / 片思いモード
- Sibling mode / きょうだいモード
- Same-year colleague mode / 同期モード
- Childhood friend mode / 幼なじみモード
- With people who share my interests mode / 同じ趣味の人といるときモード *(Layer 2 example / Layer 2例として)*

### Spatial / 空間的
- At home mode / 家にいるときモード
- At my childhood home mode / 実家にいるときモード
- At a café mode / カフェにいるときモード
- Traveling mode / 旅行中モード
- On a train or bus mode / 乗り物の中モード
- In nature mode / 自然の中モード
- In an unfamiliar city mode / 知らない街モード
- On the way home from school mode / 学校の帰り道モード

### Institutional / 制度的
- At school mode / 学校にいるときモード
- At work mode / 職場にいるときモード
- As a patient at the hospital mode / 患者として病院にいるときモード
- At a government office mode / 役所にいるときモード

### Digital / デジタル
- Fan activity mode / 推し活モード
- Performing for my followers mode / フォロワーに向けてのモード
- Streaming mode / 配信モード
- Anonymous mode / 匿名でいるときモード
- Just scrolling mode / スマホ見てるだけモード
- Online meeting mode / オンライン会議モード
- Talking with AI mode / AIと話しているときモード

*(Digital modes are subject to periodic review as technology and culture evolve.)*
*(デジタルモードは、技術・文化の変化に伴い定期的な見直しを推奨します。)*

### Temporal / 時間的
- Past self mode / むかしの自分モード *(Layer 2: childhood mode / こどものころモード, young adult mode / 青年だったころモード, etc. — user-defined / ユーザーが自由追加)*
- Imagined self mode / 想像の中の自分モード
- Wanting to speak with someone who is no longer here mode / いまはもういない人と語りたいモード *(Layer 2: user-defined / ユーザーが自由に細分化)*

*(See SAFETY_ETHICS note regarding this mode. / このモードに関するSAFETY_ETHICSの注記を参照してください。)*

### Collective / 集合的
- Being part of a team mode / チームの一員モード
- Being with everyone mode / みんなといるときモード
- Disappearing into the crowd mode / 集団の中で埋もれているときモード

## Naming principle / 命名の原則

The AI proposes mode names in the format "When I'm with 〇〇" (〇〇といるときモード) as a default.
Users can rename, shorten, or replace these at any time.
There is no correct name. The right name is the one that feels right to you.

AIはデフォルトとして「〇〇といるときモード」形式でモード名を提案します。
ユーザーはいつでも名前を変更・短縮・置き換えることができます。
正しい名前は存在しません。あなたにとってしっくりくる名前が、正しい名前です。

Mode name lists are version-managed (e.g. v1.0, v2.0) across all categories.

モード名候補リストは全カテゴリを対象にバージョン管理されます（例：v1.0、v2.0）。

## Mode card / モードカード

Each mode is represented as a card containing:

各モードはカードとして表示され、以下を含みます：

| Item / 項目 | Description / 説明 |
|------|-------------|
| Mode name / モード名 | AI-proposed or user-entered; editable at any time / AI提案またはユーザー入力；いつでも編集可 |
| 1–2 questions / 質問1〜2問 | Drawn from a rotating pool; not the same every time / プールからローテーション；毎回同じではない |
| A word to yourself / このモードの自分に一言 | Free text; always present / 自由記述；毎回固定 |

**Question pool / 質問プール（例）**
- Did you feel at ease? / 安心できましたか？
- Did you get to smile? / 笑顔になれましたか？
- Did you get to talk a lot? / たくさん話せましたか？

Questions rotate from the pool. The same question is not shown every session.

質問はプールからローテーションされます。毎回同じ質問は表示されません。

---

## Visible vs. internal modes / 表示モードと内部モード

| Type / 種類 | Count / 数 |
|------|-------|
| Visible on board / ボードに表示 | 5–7 cards |
| Internal (stored) / 内部（保存済み） | 50–100 |

Users see a curated surface. The full set of modes exists underneath.

ユーザーには厳選されたサーフェスが表示されます。モードの全体はその下に存在します。
