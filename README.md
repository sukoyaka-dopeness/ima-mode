# ima-mode / いまモード

**a tool for noticing yourself, right now.**
**いまの自分に気づくための場所。**

---

## What is this? / これは何？

ima-mode is a self-observation tool that helps you notice your relational states — not to judge them, improve them, or track them over time, but simply to see them.

ima-modeは、自分の関係性の状態に気づくためのツールです。評価するためでも、改善するためでも、記録し続けるためでもありません。ただ、見るための場所です。

You are not the same person with everyone. With your boss, with your best friend, alone at a café — these are all you. ima-mode gives each of those versions a name.

あなたは、誰といるかによって、少しずつ違います。上司といるとき、親友といるとき、カフェでひとりでいるとき——それは全部あなたです。ima-modeは、そのひとつひとつに名前をつけます。

---

## Core principles / 基本的な考え方

**No diagnosis. No advice. No scores.**
診断しません。アドバイスしません。スコアもつけません。

**AI as a transparent mirror.**
AIは透明な鏡です。キャラクターではありません。感情をシミュレートしません。

**Your data belongs to you.**
データはあなただけのものです。ローカルにのみ保存され、外部に送信されることはありません。デフォルトで1日後に自動削除されます。

**All modes belong to one self.**
すべてのモードは、ひとりの自分のものです。

---

## What ima-mode is not / ima-modeではないもの

- A mental health app / メンタルヘルスアプリではありません
- A mood tracker / 気分トラッカーではありません
- A self-improvement tool / 自己改善ツールではありません
- A monitoring system / モニタリングシステムではありません
